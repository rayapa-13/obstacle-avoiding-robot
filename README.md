This project is an Arduino-based obstacle avoiding robot using L293D motor driver and ultrasonic sensor (HC-SR04), simulated in Tinkercad.

🔧 Components Used
- Arduino Uno
- L293D Motor Driver
- 2 DC Motors
- HC-SR04 Ultrasonic Sensor
- 9V Battery

⚙️ Working

The robot moves forward and detects obstacles using ultrasonic sensor. If an object is detected within range, it stops, moves back, and turns to avoid collision.

💻 Simulation
Built and tested in Tinkercad before hardware implementation.

🚀 Future Improvements
- Servo scanning
- Bluetooth control
- Speed control using PWM
